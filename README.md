### 🎙️ AI Architectural Layout Generator · [Live Demo ↗](https://vox-assist-pearl.vercel.app)

VoxAssist converts natural language prompts and structural dimensions into full-scale **2D floorplans and 3D architectural models** — entirely powered by a custom-trained local LLM (`vox-architect` via Ollama).

**What makes it interesting:**

- 🌐 **3D Visualization:** Immediate structural parsing of 2D plans into 3D meshes via React Three Fiber
- 🐳 **Full DevOps:** Docker-compose, GitHub Actions CI (Pytest + Vitest + pip-audit), keep-alive pinger

## 🕹️ Featured Project — Pacman Reinforcement Learning

<div align="center">

<table>
<tr>
<td width="58%">

### 🧠 Approximate Q-Learning Agent · [Repo ↗](https://github.com/PuneethPujala/Pacman-using-RL)

A deep dive into **Approximate Q-Learning** — not just running an existing RL framework, but diagnosing and fixing fundamental algorithmic bugs to raise the agent's win rate from **2% → 50%** on the `smallClassic` layout.

**Key engineering contributions:**
- 🐛 **Bootstrap Fix:** Identified that `computeValueFromQValues` was hardcoded to tabular lookups — agent was blind to future rewards. Fixed polymorphically.
- 📉 **Decay Schedules:** Implemented exponential decay for ε and α: `α_t = α₀ · 0.985^t` to prevent weight oscillation
- 🔀 **Hybrid Features:** Linear distance for global objectives, non-linear inverse for local ghost threats — eliminated the "disappearing weight" bug
- 🚧 **Corridor Danger Feature:** Custom feature that learned a −971 weight for dead-end traps near active ghosts

| Agent | Win Rate | Avg Score |
|-------|----------|-----------|
| Baseline | 2% | 48 |
| After Fix | 38% | 652 |
| **Optimized** | **50%** | **997** |
