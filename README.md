## 🏥 Featured Project — CareMyMed

<div align="center">

<table>
<tr>
<td width="60%">

### 🤖 An AI-powered healthcare companion for India

CareMyMed is a full-stack mobile platform targeting **chronic disease management** for elderly users and their families. Built as a monorepo with React Native, Express.js, and a FastAPI AI service.

**Core Features:**
- 💊 Intelligent Medication Reminders & Adherence Tracking
- 🩺 AI Health Assistant (conversational, context-aware)
- 📷 OCR Prescription Scanner
- 🎙️ Voice Assistant (powered by Whisper + local LLMs)
- 👨‍👩‍👧 Family Companion Dashboard
- 📊 Health Analytics & Intelligent Insights
- 🔒 Offline-first architecture
- 🧠 Local LLM integration via Ollama
